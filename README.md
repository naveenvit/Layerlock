# 🔐 LayerLock

> Detect Early. Correlate Intelligently. Contain Before It’s Gone.

LayerLock is a Unified Cyber + AML Intelligence Platform designed to detect and prevent coordinated money laundering attempts by correlating cyber compromise signals with financial transaction behavior in real time.

This project was built as a fintech cybersecurity SaaS prototype for hackathon evaluation.

---

## 🚨 Problem Statement

Modern financial crime is increasingly driven by cyber attacks such as phishing and account takeovers.

However:

- Cybersecurity systems detect breaches.
- AML systems monitor transactions.
- These systems operate in silos.
- Suspicious transactions appear normal in isolation.
- Mule networks are only visible at the cluster level.
- Detection often happens too late — after funds are layered and dispersed.

There is no unified intelligence layer that correlates cyber compromise signals with transaction behavior in real time.

---

## 💡 Our Solution

LayerLock bridges the intelligence gap by:

- Correlating cyber events and financial transactions
- Generating a unified fraud risk score (0–100)
- Detecting mule network expansion
- Propagating risk across connected accounts
- Using Gemini AI for explainable fraud reasoning
- Triggering early containment before layering completes

The system operates in two modes:

🏦 Bank Mode – Technical intelligence & compliance view  
👤 User Mode – Simplified risk visibility & safety insights
